NCATS TRANSLATOR KNOWLEDGE.BIO LAUNCH WHAT IS IT? The NCATS Translator Initiative Knowledge.Bio web client application is an open-source project that allows its users to discover and document biomedical connections across multiple distributed "Knowledge Beacons", that is, online biomedical databases that implement the NCATS Knowledge Beacon Web Application Programming Interface. Click the FAQ(?) button in the left side bar menu to learn more about the tool and how to use it."
