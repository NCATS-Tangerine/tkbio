NCATS TRANSLATOR KNOWLEDGE.BIO LAUNCH WHAT IS IT? The NCATS Translator Initiative Knowledge.Bio web client application is an open-source project that allows its users to discover and document biomedical connections across multiple distributed "Knowledge Beacons", that is, online biomedical databases that implement the NCATS Knowledge Beacon Web Application Programming Interface.
1 enter a keyword 2 refine your search 3 add to map 4 arrange your concept map 5 find related evidence (e.g. journal citations) 6 search public library of concept maps USE CASE   Here is an example of what you can discover.
